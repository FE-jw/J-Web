# **Git 명령어 정리**

## **git add**

## **git commit**

## **git push**

## **git branch**
브랜치 목록을 조회한다.

## **git switch [my-branch]**
'my-branch'로 브랜치를 변경한다.

## **git merge [my-branch]**
main 브랜치에서 실행하며, my-branch 내용을 merge한다.

## **git push origin [my-branch]**
'my-branch'라는 원격 브랜치를 생성한다.

## **git push origin -d [my-branch]**
'my-branch'라는 원격 브랜치를 제거한다.
